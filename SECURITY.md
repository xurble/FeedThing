# Security Policy

## Supported Versions

This is a hobby project, only the latest versions are supported.

| Version | Supported          |
| ------- | ------------------ |
| 3.7     | :white_check_mark: |
| < 3.7   | :x:                |

## Reporting a Vulnerability

Vulnerabilites can be reported via email to g@xurble.org.  Please include FeedThing Security
in the subject-line.
